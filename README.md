# Personal Portfolio & Blog

My personal portfolio and blog where I share projects and write about AI/ML engineering.
## 🛠️ Technologies used

- ⚡️ Astro.js (^4.10.1)
- 🔥 React.js
- 🎨 Tailwind CSS
- 🔍 TypeScript

## Installation

1. Clone the repository
```bash
git clone git@github.com:YOUR_GITHUB_USERNAME/my-portfolio-blog.git
```

2. Go to the project directory:
```bash
cd my-portfolio-blog
```

3. Install packages
```bash
npm install
```

4. Run locally in development mode
```bash
npm run dev
```
